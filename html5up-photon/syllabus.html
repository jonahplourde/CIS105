Course Syllabus
CS105 Introduction to Programming

Comprehensive 12-week curriculum covering
programming fundamentals and practical applications

Course Overview

Course Code: CS105
Course Name: Introduction to Programming
Duration: 12 weeks (24 sessions, 90 minutes each)
Prerequisites: None - open to beginners
Language: Python 3.x

Course Objectives

By the end of this course, students will be able to:

  • Understand fundamental programming concepts and principles
  • Write, debug, and test Python programs
  • Use data structures (lists, dictionaries, tuples) effectively
  • Implement control structures (loops, conditionals)
  • Create functions and understand scope
  • Work with files and handle exceptions
  • Apply object-oriented programming basics
  • Use libraries and APIs in projects
  • Solve problems using algorithmic thinking

Weekly Schedule

Weeks 1-2: Getting Started

  • Setting up your development environment
  • Introduction to Python and basic syntax
  • Variables, data types, and operators
  • Input/output and string manipulation
  • Project: Simple calculator program

Weeks 3-4: Control Structures

  • Conditional statements (if/elif/else)
  • Loops (for and while)
  • Nested loops and loop control
  • List comprehensions
  • Project: Number guessing game

Weeks 5-6: Functions and Modules

  • Defining and calling functions
  • Parameters and arguments
  • Return values and scope
  • Lambda functions
  • Importing and using modules
  • Project: Library management system

Weeks 7-8: Data Structures

  • Lists, tuples, and dictionaries
  • List methods and operations
  • Dictionary operations and iteration
  • Sets and their applications
  • Nested data structures
  • Project: Data analyzer for CSV files

Weeks 9-10: File Handling and Error Management

  • Reading and writing files
  • Working with different file formats
  • Exception handling (try/except)
  • Debugging techniques
  • Logging and error messages
  • Project: File processing application

Weeks 11-12: Object-Oriented Programming & Final Project

  • Classes and objects
  • Methods and attributes
  • Inheritance basics
  • Introduction to web development (Flask)
  • Working with APIs
  • Final Project: Interactive web application

Grading Policy

  • Weekly Assignments: 40%
  • Projects: 35%
  • Midterm Project: 15%
  • Final Project: 10%

Grading Scale:
A: 90-100% | B: 80-89% | C: 70-79% | D: 60-69% | F: Below 60%

Required Materials

  • Computer with internet access (Windows, Mac, or Linux)
  • Python 3.x (free download - installation guidance provided)
  • Text editor or IDE (VS Code recommended)
  • All course materials provided online

Policies

Attendance

Regular attendance is expected. Students missing more than 4 sessions may be subject to academic review.

Late Assignments

Assignments submitted late will receive a 10% deduction per day, up to 3 days. After 3 days, assignments will not be accepted without prior approval.

Academic Integrity

All work must be your own. Collaboration is encouraged for learning, but copying code is strictly prohibited. Citation of resources is required when using external code.
